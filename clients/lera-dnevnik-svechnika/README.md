# Колесо фортуны Леры

Файл `loader.js` подключает виджет одной строкой и вставляет его прямо в страницу клиента, без iframe. Кнопки остаются обычными элементами страницы и сохраняют метки:

- `data-track="wheel-spin"` — кнопка «Крутить»
- `data-track="wheel-get-prize"` — кнопка «Забрать приз»

Код для Tilda лежит в `install-code.txt`.
